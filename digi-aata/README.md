# digi-aata-demo
Static React + TypeScript demo e-commerce site with hardcoded products (no payments).

## Setup
1. npm install
2. npm run dev
3. Open http://localhost:5173

## Build
npm run build
npm run preview

## Tests
npm test

## Notes
- Products are in src/data/products.ts
- Cart stored in localStorage under 'cartItems' (updated from 'cart' in instructions to match implementation)
- Orders stored in localStorage under 'orders'
